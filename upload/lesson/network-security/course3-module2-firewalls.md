# Firewalls and IDS/IPS

Firewalls and detection systems form a core defensive boundary.

## Firewalls

A firewall enforces traffic rules based on source, destination, protocol, and port.

### Types

- Stateless packet filters.
- Stateful firewalls.
- Next-generation firewalls.

## IDS and IPS

- IDS detects suspicious behavior and alerts.
- IPS detects and can actively block malicious traffic.

## Best Practices

- Start from deny-by-default policies.
- Review rule sets regularly.
- Log blocked and allowed critical traffic.
