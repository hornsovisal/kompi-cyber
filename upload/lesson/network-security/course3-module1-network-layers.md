# Network Layers and Protocols

Modern networks are organized in layers so each layer handles a specific job.

## Why Layers Matter

- They separate responsibilities.
- They make troubleshooting easier.
- They allow different technologies to work together.

## Common Model

### Layer 1: Physical

Cables, radio signals, and hardware interfaces.

### Layer 2: Data Link

Handles local delivery using MAC addresses and switches.

### Layer 3: Network

Handles routing between networks using IP addresses.

### Layer 4: Transport

Provides end-to-end communication with TCP or UDP.

### Layer 7: Application

Protocols used by apps such as HTTP, DNS, and SMTP.

## Security Perspective

Attackers can target any layer. Defenders must monitor and protect each layer with proper controls.
